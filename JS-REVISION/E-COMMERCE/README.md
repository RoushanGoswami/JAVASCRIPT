# 🛒 E-COMMERCE PROJECT 

## 🧠 Overview
📸 : https://drive.google.com/file/d/1z2bl3u98DzEADj6zs12DMtSNpoBunVlC/view?usp=sharing
This project is a **vanilla JavaScript-based E-Commerce application** created as part of a revision series. It focuses on core frontend development concepts such as DOM manipulation, event handling, and client-side state management.

It is a **learning-focused project** that simulates an online shopping experience without using frameworks or backend services.

---

## 🎯 Objective

The purpose of this project is to demonstrate:

* Strong understanding of **JavaScript fundamentals**
* Ability to build **interactive UI using DOM manipulation**
* Implementation of **basic e-commerce logic**
* Handling of **cart operations using client-side storage**

---

## 🧱 Project Structure

* **HTML** → Page structure and layout
* **CSS** → Styling and UI design
* **JavaScript** → Application logic and interactivity

The project follows a **frontend-only architecture** with no backend integration.

---

## 🛍️ Key Features

### ✅ Implemented Features

* Product listing display
* Add to cart functionality
* Cart item management (add/remove/update)
* Quantity handling (if implemented)
* Total price calculation
* Persistent cart using localStorage (if used)

---

### ❌ Missing Features (Production Level Gaps)

* No backend/API integration
* No user authentication system
* No database (MongoDB/Firebase/etc.)
* No payment gateway integration
* No order tracking system
* No admin dashboard

---

## ⚙️ Technical Strengths
