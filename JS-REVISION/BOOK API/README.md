# 🎯 Quiz App

A simple and interactive **Quiz Application** that challenges users with **10 multiple-choice questions (MCQs)**. Each question carries **1 mark**, and the entire quiz must be completed within **60 seconds**.

---

## 🚀 Features

* 📝 **10 MCQs** per quiz session
* ⏱️ **60-second timer** for the entire quiz
* ✅ **1 mark per correct answer**
* ❌ Instant feedback on selected answers (optional feature)
* 📊 Final **score display at the end**
* 🔄 Option to **restart the quiz**

---

## 🕹️ How It Works

1. The user starts the quiz.
2. A timer of **60 seconds** begins immediately.
3. The user answers **10 multiple-choice questions**.
4. Each correct answer earns **1 point**.
5. Once:

   * All questions are answered, OR
   * Time runs out

   👉 The quiz ends automatically.
6. The final score is displayed.

---

## 🧠 Scoring System

| Criteria            | Marks |
| ------------------- | ----- |
| Correct Answer      | +1    |
| Incorrect Answer    | 0     |
| Unanswered Question | 0     |

---

## ⏳ Time Limit

* Total time: **60 seconds**
* The timer runs continuously.
* Quiz auto-submits when time expires.
