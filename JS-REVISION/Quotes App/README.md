# 🚀 Futuristic Quote App

A beautifully designed **Futuristic Quote Generator Web App** built with modern UI principles, smooth animations, and an interactive user experience.

This project is not just a simple quote generator — it's a **fully functional quote management system** with like, navigation, and favorites handling.

---

## 🌟 Features

✨ **Dynamic Quote Display**

* Displays quotes with author names
* Smooth animated transitions between quotes

⏮️ **Previous Button**

* Navigate back to previously viewed quotes

⏭️ **Next Button**

* Instantly load a new quote

❤️ **Like / Unlike System**

* Like your favorite quotes
* Toggle like (add/remove functionality)

📌 **Favorites Section**

* Save liked quotes
* View them separately in a dedicated page

🎨 **Futuristic UI Design**

* Dark theme with neon accents
* Glassmorphism effects
* Smooth hover and click animations
* Responsive layout

## ⚙️ How It Works

1. Quotes are stored in JavaScript
2. Clicking **Next** loads a new quote
3. Clicking **Previous** shows the last quote
4. Clicking **Like ❤️**:

   * Adds quote to favorites
   * Clicking again removes it
5. Favorites page displays all liked quotes
